# Store Listing Draft

## App Name

PHRIT

## Subtitle

One photo. One sentence. One day.

## Short Description

PHRIT helps you answer a daily question with one image and one short sentence.

## Description

PHRIT is a quiet social journal for daily moments. Each day, answer one question with a photo and a sentence. See what others felt, leave a small reaction, save posts, and keep your own archive of days.
