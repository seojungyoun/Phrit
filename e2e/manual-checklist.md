# E2E Manual Checklist

Run this checklist on a real iOS or Android device after `npm run start`.

1. Sign up with email.
2. Confirm email if Supabase email confirmation is enabled.
3. Sign in.
4. Confirm today question appears.
5. Create a post with one photo and one caption.
6. Confirm the post appears in the feed.
7. React to another user's post.
8. Open post detail and add a comment.
9. Save and unsave a post.
10. Report another user's post.
11. Block another user and confirm their posts disappear.
12. Edit username and bio.
13. Open saved posts from profile.
14. Sign out and sign back in.
15. As admin, create tomorrow's question.
16. As admin, review a report.
